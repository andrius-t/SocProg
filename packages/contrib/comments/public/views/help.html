You can find your package in /packages/contrib/comments. This is the first package contribution to MEAN.
To include comments for now, you should just include the necessary file
For example on -: articles/views/view.html or articles/views/list.html
After {{article.content}}, insert the folowing 

      <div ng-if="article !== undefined" ng-include="'comments/views/index.html'" data-ng-controller="CommentsController" data-ng-init="findComments(article, 2)"/>
